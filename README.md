# Steps to run the project

- Make sure to have node.js installed

  ```bash
  $ git clone https://github.com/ratika-12/movie-web-app.git
  $ cd movie-web-app
  ```


- ## Frontend
 1. ```cd movie-frontend```
 2. ```npm install && npm run start```

- ## Backend
1. ```cd movie-backend```
2. ```npm install &&  nodemon server.js```
